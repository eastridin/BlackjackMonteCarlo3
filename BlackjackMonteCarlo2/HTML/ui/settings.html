Settings menu

  • The game tab allows you to modify game settings.
  • AI iterations means how many Monte Carlo Tree Searches will take place to make a decision.
  • MCTS iterations means how many iterations an individual Monte Carlo Tree Search will perform before being completed.
  • The UCT constant is the value that the user can input which will affect how nodes are selected by the MCTS. It is a constant used in the formula used to select nodes to explore in a MCTS.
  • If the settings menu was openend within a blackjack game, the players tab will be available. The players tab allows you to modify each player's settings, including the option to modify their balance or change its player type to an AI or user.
