Tree Visualiser

  • The tree visualiser is displayed whenever an AI instance of a player makes a move using a Monte Carlo Tree Search.
  • This display consists of a tree on the left of the screen. The tree that is shown is the tree of the chosen iteration number of the Monte Carlo Tree Search of the chosen move of the chosen player.
  • The displayed tree can be changed by utilising the controls on the right, where you can either use the arrows to move over to the next iteration of the Monte Carlo Tree Search, or you can manually input the iteration number in the text box.
  • Additionally, you can select which AI player and which move to view the trees of.
  • The display on the right also gives information of the overall decision process, giving you the number of tree searches that took place, the best move found by the currently chosen tree search, and then the overall results of all the tree searches.
  • Each node in the tree has information inside of it. The numbers in the form x/y represent the value of that node (x) and the number of visits to that node during the tree search (y)
  • Below the numbers, the previous move made by that node's game state is displayed to the user.
  • The displayed tree can be interacted with by the user:
  • Single clicking on a node will hide all the children from that node from the user. This allows the user to concentrate on certain aspects of the tree at once.
  • Double clicking on a node will open a new window which gives the user information about that particular node.
