Main Menu

  • The main menu consists of a navigation menu on the left which consists of 4 different options: New Game, Load Game, Simulate and Settings.
  • The New Game menu consists of parameters for a new game which include the number of players, the type of each player and the balance of each player. You can then initiate a new game with these parameters by pressing the new game button.
  • The Load Game option will open up a file explorer window in the current directory and you can then select a game save to load.
  • The Simulate menu consists of options for simulating a number of blackjack games with the AI. You can then start this process and view the resulting balance on the right hand side.
  • The settings menu will open in a new window and allow you to modify various aspects about the program.
